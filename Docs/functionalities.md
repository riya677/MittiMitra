# Project Functionalities

This document lists all the modules and their functionalities in the project, along with purpose, input, and output.

| Module         | Functionality                         | Purpose                                   | Input                   | Output                   |
|----------------|---------------------------------------|-------------------------------------------|------------------------|-------------------------|
| Soil Analysis  | AI model-based NPK/pH detection        | Analyze soil nutrients and pH             | Soil sample image       | NPK levels, pH value    |
| Weather        | Show real-time weather & location     | Provide current weather information       | Location (GPS)         | Temperature, humidity   |
| Recommendations| INM & fertilizer guidance             | Suggest proper fertilizers                | Soil analysis results   | Fertilizer recommendations |
| Camera         | Capture soil sample image             | Take picture of soil for analysis         | User action            | Soil image              |
| Profile        | Farmer info, preferences              | Manage farmer details                      | Farmer input           | Stored profile data     |
| History        | Store & view previous analyses        | Keep track of past soil tests             | N/A                    | Past analysis records   |
| Notifications  | Alerts, reminders                     | Notify farmers about tasks and updates    | N/A                    | Alert message           |
| Offline/Sync   | Data backup and sync                  | Enable offline usage and syncing          | Updated data           | Synced data             |
| Multilingual   | Regional language support             | Support multiple languages                | Selected language      | App displayed in language |
| Analytics      | Track app usage                        | Monitor app usage statistics              | User actions           | Usage reports           |
